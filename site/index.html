Seven Safety Gates

Every filing passes through seven independent safety checks before touching CM/ECF.

  1. PDF Validation — File must be valid PDF, under 100MB, searchable, unencrypted, no fillable forms.
  2. Redaction Scan — AI + regex scanning for unredacted personal identifiers under Rule 5.2.
  3. Event Code Verification — AI confirms the selected event code matches the document content.
  4. Completeness Check — Verifies required attachments (proposed orders, certificates of service).
  5. Attorney Review — Full filing summary displayed. You must type CONFIRM to proceed.
  6. Final Submit Watchdog — Screenshot of CM/ECF confirmation page shown. You must type YES.
  7. Receipt Capture — Filing confirmation saved as HTML + logged to audit database.
