Seven safety gates.

Every filing passes through seven checks before touching CM/ECF.

1
PDF Validation — valid, under 100MB, searchable, no encryption.
2
Redaction Scan — AI + regex for Rule 5.2 personal identifiers.
3
Event Verification — AI confirms event code matches document.
4
Completeness — checks for missing attachments, certificates.
5
Attorney Review — full summary. You type CONFIRM.
6
Final Watchdog — screenshot of CM/ECF page. You type YES.
7
Receipt Capture — confirmation saved, logged to audit trail.
